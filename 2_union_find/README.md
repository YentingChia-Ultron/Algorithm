Union Find
===
## Overview

1. Given N nodes
1. **union** some of nodes
1.  **determine** whether any **two nodes** are **connected**

#### My Methods
* Quick Find
* Quick Union
* Weighted Quick Union

---
## Cost


|      Algorithm       | Woest-Case time | 
|       :--------:     |   :--------:    | 
|      Quick Find      |      M N        | 
|      Quick Union     |      M N        | 
| Weighted Quick Union |   N + M log N   | 
**M operations of N nodes**
